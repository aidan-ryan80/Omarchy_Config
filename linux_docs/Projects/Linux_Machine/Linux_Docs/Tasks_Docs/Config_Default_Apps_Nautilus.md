### Goal:
##### Configure default apps for file types in the nautilus file explorer.

- Example file types =>.py, .txt, .html, .xml, .css, etc...
- The above examples above should be configured to nvim
- Fix opening files from nautilus to nvim

### Save for Later:
- Apps located in:
```bash
cd ~/.local/share/applications
```
- Defaults in:
```bash
cd ~/.config/mimeapps.list
```
- Search for GPT Chat with by searching for "mimeapps" or "nautilus"
