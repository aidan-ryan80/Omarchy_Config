### Goal:
##### To figure out what the default keyring popup is for on my machine

### Observations:
- Prompt appears on reboot for browsers
- Seems to happen when changing Theme sometimes with Ctrl + Shift + Super + Space
- Happens when using Git
	- when using git to push changes right after restarting machine
	- when cloning a remote repo\

##### Answer:
Seems to be a place to secure credentials or passwords for some applications that need to use them to function such as browsers and git. Omarchy already made it to automatically be done in the background on startup.
